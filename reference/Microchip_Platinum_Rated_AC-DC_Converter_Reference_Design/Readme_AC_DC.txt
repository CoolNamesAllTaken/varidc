The 720W Platinum-Rated AC/DC Reference Design Package contains the following:
- Application Note
- Schematic and Layout Files (CADSTAR)
- Source Code
- Bill of Materials
- Demo Instructions
- Efficiency Measurement Guidlines

Getting Started:
See 720W Platinum Rated AC/DC Overview for features, specifications, and a brief overview of the system and software. Refer to Demo Instructions and Efficiency Measurement Guidlines for instructions on system setup and performance measurements.


Known Issues:
AC frequency lockout detection is not implemented. The specification calls for operation between 47Hz and 63Hz but the system does not shut down when operation occurs outside of the specifications.
